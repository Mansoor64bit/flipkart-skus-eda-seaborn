# Flipkart SKU Dataset EDA using Seaborn

## Overview

This project performs Exploratory Data Analysis (EDA) on a Flipkart SKU dataset using Python and Seaborn.

The objective was to analyze product data, pricing patterns, category trends, and other business-related insights through visualization and statistical analysis.

---

## Tools Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

---

## Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Product Category Analysis
- Brand Analysis
- Pricing Analysis
- Correlation Analysis
- Data Visualization

---

## Dataset Features

The dataset contains information such as:

- Product Name
- Category
- Brand
- Price
- Rating
- Discount
- Product Attributes

---

## Key Insights

- Identified product categories with the highest SKU concentration.
- Analyzed pricing patterns across categories.
- Examined brand-level product distribution.
- Visualized relationships between price, ratings, and discounts.
